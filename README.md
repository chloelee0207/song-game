# 🎤 Sing the Word!

A tiny one-button party game for picnics and get-togethers.

**How to play:** Press the button, think of a song that contains the word it
shows, and the first person to sing it wins a point!

## Run it

Just open `index.html` in any web browser — on a laptop or a phone. No install,
no internet needed once the page is loaded.

The whole game is a single self-contained HTML file, so you can also drop it
onto any static host (GitHub Pages, Netlify, etc.) if you want a shareable link.
